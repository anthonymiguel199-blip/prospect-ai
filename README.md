# 🔎 ProspectAI — Lead Generation Engine

Este é um motor de busca de leads em tempo real desenvolvido para a **Nexux Marketing Digital**.

## 🚀 Tecnologias
- HTML5 / CSS3 / JavaScript (Vanilla)
- Node.js (Vercel Serverless Functions)
- API do Apify (Google Maps Scraper)

## ⚙️ Funcionalidades
- Busca de leads por nicho e cidade.
- Cálculo automático de Lead Score.
- Exportação de leads qualificados.
